## Identified unsafe routes

### Route A: via `tateAcyclicity_gluing_via_refinement` + `refines_by_standard_cover`

**Unsafe reasons:**
- `refines_by_standard_cover` currently threads `hZavyalov` hypothesis.
- Discharging `hZavyalov` needs Wedhorn Prop 7.14 (full adic Nullstellensatz) — not in project.
- Alternative: apply `Cor 7.32` which needs `MulArchimedean` — also leaks.

**Verdict**: AVOID until Prop 7.14 lands, OR route doesn't go through StandardCover.

### Route B: "direct partition of unity" (comments at LaurentRefinement.lean:3728)

Described as:
1. Span-top: `Ideal.span {canonicalMap(D.s)} = top` in `presheafValue C.base`
2. Surj: `f D = r_D / sD^n_D` via `IsLocalization.Away.surj`
3. Uniform exponent
4. Numerator compatibility
5. Partition of unity: `∑ c_D · sD^N = 1`
6. Global section: `x = ∑ c_D · r''_D`

**Unsafe reasons:**
- **Span-top (step 1)** requires the Spa-point argument for every prime containing `canonicalMap(D.s)` — essentially Wedhorn Prop 7.14 content.
- Still requires `restrictionMapHom_surj` (PresheafTateStructure:1207) and `restrictionMapHom_injective` (same file:1238), both with their own sorries.

**Verdict**: Partly blocked on Prop 7.14 content (span-top step) + Cor 8.32 chain (Part 1 dependency already tracked).

### Route C: `laurentCover_gluing_presheaf` via `row3_exact` (Wedhorn Lemma 8.33)

**Usable for 2-element Laurent cover** (`{laurentPlus, laurentMinus}`), not a general rational cover.

Requires:
- `laurentOverlapBridge_exists_compatible` (Example 6.39 — 400-600 lines, still open)
- All the plus/minus bridges (all closed now ✅)

For general rational covers, this reduces to Route A or a custom refinement.

**Verdict**: Only handles the 2-element case. Doesn't give full tateAcyclicity Part 2.

## Safe route proposed: Route D — bridge through the Laurent-cover base case

**Core idea**: Prove tateAcyclicity Part 2 for the SPECIFIC case of a 2-element Laurent cover (Route C), then EXTEND to general rational covers VIA REFINEMENT — but the refinement step itself uses `separation_of_finer_rational` (already proved, sorry-free) and `gluing_of_finer_rational` (proved, sorry-free).

**Step-by-step plan:**

### D1. Prove `laurentCoverGluing` (2-element Laurent case)

Using Route C (Wedhorn Lemma 8.33 / `laurentCover_gluing_presheaf_viaRow3`), which needs:
- Plus/minus bridges ✅ (all proved)
- `laurentOverlapBridge_exists_compatible` — the overlap bridge (Example 6.39)

**Status**: blocked on Example 6.39. Closing it closes this step.

### D2. Extend to general rational covers via iterated refinement

A general rational cover `C` can be refined to a product of Laurent covers by ITERATED application:
- Given `C = {D_1, ..., D_n}` (finite), pick an element `f` separating some pair.
- Refine by Laurent cover at `f`.
- Iterate on each piece.

This is Wedhorn Lemma 8.34, but with the CONCRETE recursion on `|C|` rather than an abstract Zavyalov construction.

**Pros**: doesn't need hZavyalov / Cor 7.32 / MulArchimedean.
**Cons**: each iteration uses `separation_of_finer_rational` and `gluing_of_finer_rational` (already proved), and the induction base case uses D1 (above).

**Status**: Can be attempted once D1 closes. Estimated ~200 lines.

### D3. Close `tateAcyclicity` Part 2 via D1 + D2

Final mechanical assembly combining the Laurent-cover base + refinement induction.
