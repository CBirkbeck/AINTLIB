## Statement (Silverman III.2.3.1)
A function `f ∈ K̄(E)` satisfies `f ∘ [-1] = f` (i.e. `f(P) = f(-P)`) iff
`f ∈ K̄(x)` (i.e., `f` lies in the subfield generated by `x` over `K̄`).

## Notes
- Uses `[K(E) : K(x)] = 2` (T-III-3-002).
- The negation map sends `(x,y) ↦ (x, -y - a₁ x - a₃)`. Functions invariant
  under this map are exactly polynomials in `x` (and `y² + a₁ x y + a₃ y`,
  which equals `x³ + a₂ x² + a₄ x + a₆ ∈ K[x]`).
