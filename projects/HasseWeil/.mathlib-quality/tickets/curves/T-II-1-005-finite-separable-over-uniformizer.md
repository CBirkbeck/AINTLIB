## Statement (Silverman II.1.4)
Let `C/K` be a smooth curve, and let `t ∈ K(C)` be a uniformizer at some
nonsingular point `P ∈ C(K)`. Then `K(C)` is a finite separable extension of `K(t)`.

## Why BLOCKED

The full ticket has two distinct sources of difficulty, only one of which is
about the Weierstrass-specific structure; the other needs substantial new
algebraic-geometric machinery.

### Blocker 1 — Separability via Kähler differentials

For a **smooth** curve and a uniformizer `t` at a smooth point, `dt ≠ 0` in
`Ω_{K(C)/K}` at `P`. This is equivalent to `K(C)/K(t)` being separable. The
standard proof goes:

1. `dim_K(C) Ω_{K(C)/K} = 1` (rank-one follows from the smoothness + dimension-1).
2. A differential `dt` is nonzero at a point `P` iff `t` is a uniformizer at `P`.
3. Therefore `dt ≠ 0` generically, which gives `K(C)/K(t)` separable.

Our project does have `HasseWeil/Curves/Differentials.lean` stating
`Ω[K(C)⁄F]`, but does not prove it is one-dimensional. That fact is ticket
`T-II-4-002`, itself an open substantial ticket.

In **characteristic zero** or when the residue field is perfect and the curve
is not the `p`-th power of another curve, `K(C)/K(x)` is automatically separable
and the only content is the "arbitrary t" generalization below. But the ticket
statement claims separability in arbitrary characteristic, which **requires**
the uniformizer-at-smooth-point hypothesis and the Kähler-differential argument.

### Blocker 2 — Arbitrary transcendental t

The statement of II.1.4 uses an **arbitrary** uniformizer `t` at any smooth
`K`-rational point. The existing infrastructure in `HasseWeil/Basic.lean`
gives finiteness only for the specific coordinate function `x = mk W (C X)`:

```lean
-- HasseWeil/Basic.lean:309
theorem WeierstrassCurve.degree_functionField_over_kx :
    Module.finrank (FractionRing F[X]) W.toAffine.FunctionField = 2
-- and the private helper
private theorem mulByInt_finrank_functionField_eq_two :
    Module.finrank (FractionRing F[X]) W.toAffine.FunctionField = 2
```

(Both currently under `[W.toAffine.IsElliptic]`; the IsElliptic assumption is
*not used* in the proof and could be dropped, but that is a separate cleanup.)

To generalize from `t = x` to arbitrary uniformizer `t`, one of the following
routes must exist:

a. **Luroth-style**: show that `K(t) ⊂ K(C)` is isomorphic to the rational
   function field `K(X)` as an `F`-algebra, then transport the finiteness.
   This needs `RatFunc.Luroth` applied to the correct submodule.

b. **Trace/norm argument**: since `K(C)` is algebraic of transcendence degree 1
   over `K`, any transcendental `t ∈ K(C)` makes `K(C)/K(t)` algebraic; combined
   with finite generation (of the Weierstrass coordinate ring), one gets
   finiteness. This requires `Algebra.IsAlgebraic` + finite generation for `K(t)
   ↪ K(C)`.

Mathlib has `IntermediateField.adjoin` and `Polynomial.aeval` — enough to state
"the subfield `F(t) ⊂ K(C)` generated by `t`". It does **not** have a ready-made
theorem of the form "if `α` is transcendental in an algebraic-of-transcendence-
degree-1 extension, then the extension is finite over `F(α)`".

### Minor issue in ticket formulation

The acceptance-criteria stub uses `Polynomial.adjoin F {t}`, which is not a
mathlib name. The intended object is either `Algebra.adjoin F {t}` (the
`F`-subalgebra, isomorphic to `F[X]` when `t` is transcendental) or
`IntermediateField.adjoin F {t}` (the subfield `F(t)`). The finite separability
statement wants the subfield: `Algebra.IsSeparable (IntermediateField.adjoin F
{t} : Subfield _) C.FunctionField` (modulo coercion fiddling).

## Acceptance criteria (revised, BLOCKED)

```lean
namespace HasseWeil.Curves

/-- For a smooth plane curve C/K, K(C) is a finite separable extension of the
    subfield F(t) ⊂ K(C) generated by a uniformizer t at a K-rational smooth
    point. Reference: Silverman II.1.4. -/
theorem finite_separable_of_uniformizer (C : SmoothPlaneCurve F)
    (P : C.RationalPoint) (t : C.FunctionField) (ht : Uniformizer C P.toSmooth t) :
    haveI : Algebra (IntermediateField.adjoin F ({t} : Set C.FunctionField))
      C.FunctionField := inferInstance
    Algebra.IsSeparable (IntermediateField.adjoin F ({t} : Set C.FunctionField))
      C.FunctionField ∧
    Module.Finite (IntermediateField.adjoin F ({t} : Set C.FunctionField))
      C.FunctionField

end HasseWeil.Curves
```

## What would unblock this

A viable path is the following split into three sub-tickets:

1. **T-II-1-005a**: Prove `K(C)` finite of rank 2 over `Frac F[X]` for
   `SmoothPlaneCurve` (no IsElliptic hypothesis). This is a ~10-line theorem
   that lifts the existing `WeierstrassCurve.degree_functionField_over_kx`
   by dropping the IsElliptic assumption (the proof doesn't use it).

2. **T-II-1-005b**: Prove that for any transcendental `t ∈ K(C)`, the subfield
   `F(t)` is `F`-isomorphic to `RatFunc F`. Then transport 005a to get
   `K(C)/F(t)` finite. Needs `RatFunc.Luroth` + `IntermediateField` API.

3. **T-II-1-005c** (needs T-II-4-002 dim-one differentials): prove separability
   via `dt ≠ 0` at the uniformizer point.

Then `finite_separable_of_uniformizer` follows by combining 005b + 005c.
