## Statement
For a curve `C` over a perfect field `K` of characteristic `p > 0` and `q = p^r`,
define `C^(q)` as the curve whose homogeneous ideal is generated by `f^(q)` for
each generator `f` of `I(C)`, where `f^(q)` raises each coefficient to the q-th
power. The **q-power Frobenius morphism** is

```
φ : C → C^(q),  [x_0, ..., x_n] ↦ [x_0^q, ..., x_n^q]
```

For `K = F_q`, we have `C^(q) = C` and `φ : C → C` is the Frobenius endomorphism.
